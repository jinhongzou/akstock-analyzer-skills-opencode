# Stock Analyzer

**功能**: 分析个股行业归属、估值水平，输出5维度评分卡 + 综合估值判定  
**数据源**: 东方财富 (akshare)

---

## 使用方式

**OpenCode 调用**:
```
skill(name="stock-analyzer")
```

**命令行运行**:
```bash
python .opencode/skills/core/src/skills/stock-analyzer/main.py <股票代码>
```

### 示例

```bash
python .opencode/skills/core/src/skills/stock-analyzer/main.py 600519   # 贵州茅台
python .opencode/skills/core/src/skills/stock-analyzer/main.py 000651   # 格力电器
python .opencode/skills/core/src/skills/stock-analyzer/main.py 002594   # 比亚迪
```

## ⭐ 你的评估任务：5维度评分卡（必须执行）

> main.py 只输出原始数据。你必须基于以下评分卡独立计算估值评分。

### 评分卡（5维度，每个维度最高10分，总计50分）

每个维度按以下规则打分。所有数据均从 main.py 输出行读取：

| # | 维度 | 数据来源（main.py具体行） | 判定条件 | 得分 |
|---|------|-------------------------|---------|:----:|
| 1 | **PE估值水平** | `市盈率(TTM): XX` | < 8（极端低估） | 10 |
|   |      |                         | 8-15（明显低估） | 8 |
|   |      |                         | 15-25（合理区间） | 5 |
|   |      |                         | 25-40（偏高） | 3 |
|   |      |                         | > 40（明显高估） | 1 |
| 2 | **PB资产质量** | `市净率: XX` | < 1.0（破净，极端低估） | 10 |
|   |      |                         | 1.0-2.0（低估） | 8 |
|   |      |                         | 2.0-4.0（合理） | 5 |
|   |      |                         | 4.0-8.0（偏高） | 3 |
|   |      |                         | > 8.0（高估） | 1 |
| 3 | **股息回报率** | `股息率(TTM): XX%` | > 6%（极高股息） | 10 |
|   |      |                         | 4-6%（高股息） | 8 |
|   |      |                         | 2-4%（中等股息） | 5 |
|   |      |                         | 1-2%（低股息） | 3 |
|   |      |                         | < 1%（微/无股息） | 1 |
| 4 | **EPS盈利强度** | `每股收益: XX` | > 2.0元（强盈利） | 10 |
|   |      |                         | 1.0-2.0元（良好） | 8 |
|   |      |                         | 0.5-1.0元（一般） | 5 |
|   |      |                         | 0-0.5元（微利） | 3 |
|   |      |                         | < 0（亏损） | 0 |
| 5 | **近期市场表现** | `涨跌幅: XX%` | 上涨 > 3%（强势） | 10 |
|   |      |                         | 上涨 0-3%（偏强） | 8 |
|   |      |                         | 微跌 -3%-0%（偏弱） | 5 |
|   |      |                         | 下跌 -3%~-8%（弱势） | 3 |
|   |      |                         | 下跌 < -8%（恐慌） | 1 |

### 综合判定

| 总分（满分50） | 判定 | 含义 | 操作建议 |
|:-------------:|:----:|:----|:--------|
| **40-50分** | 🟢 **极端低估** | 各项指标全面低估，安全边际极大 | **强烈买入** — 可重仓 |
| **30-39分** | 🟢 **明显低估** | 多个维度指向低估，安全边际充足 | **买入** — 正常仓位 |
| **20-29分** | 🟡 **合理偏低** | 估值合理略偏低估，无明显泡沫 | **持有/逢低加仓** — 等待更好价格 |
| **15-19分** | 🟠 **合理偏高** | 估值在合理区间上沿 | **持有/减仓** — 降低仓位 |
| **10-14分** | 🔴 **明显高估** | 多个维度指向高估 | **卖出/减仓** — 逐步兑现 |
| **0-9分** | 🔴 **极端高估** | 全面高估或业绩恶化 | **卖出/回避** — 离场观望 |

### 评分示例

```
输入数据:
  市盈率(TTM): 7.46 → PE评分: 10分（<8，极端低估）
  市净率: 1.43 → PB评分: 8分（1.0-2.0，低估）
  股息率(TTM): 7.70% → 股息评分: 10分（>6%，极高）
  每股收益: 1.09元 → EPS评分: 8分（1.0-2.0，良好）
  涨跌幅: -1.65% → 表现评分: 5分（微跌-3%-0%）
  ————————————————————————
  总分: 41/50 → 🟢 极端低估 → 强烈买入
```

## 行业背景参考

你应结合 `所属行业` 字段做定性补充判断：

| 行业类型 | PE特点 | PB特点 | 股息特点 |
|---------|--------|--------|---------|
| 银行/金融 | 通常低PE（5-8倍） | PB常<1 | 股息较高 |
| 消费品（成熟） | 15-25倍合理 | 2-5倍 | 稳定分红 |
| 科技/成长 | 高PE常见（30-100+） | 5-20倍 | 通常低息 |
| 周期股（钢铁/煤炭） | 高PE时是顶点，低PE时是底部 | PB<1可关注 | 波动大 |
| 公用事业 | PE稳定（15-25） | PB 1-2 | 股息稳定 |

> **注意**：以上是背景参考，不是硬性评分标准。评分按上面5维度评分卡执行，行业背景仅用于定性判断"当前PE在行业中是否合理"。
