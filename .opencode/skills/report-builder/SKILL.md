# Report Builder - 投资分析报告生成指南

**功能**: 调用8个分析Skills获取数据，基于统一模板生成投资分析报告  
**数据源**: akshare + Tushare Pro  
**模板**: guides/unified-template.md

---

## 🔶 重要原则

**所有数据必须严格按 Skills 获取到的原始数据提供，不要修改、遗漏或添加假设数据。**

- 财务数据：使用 `financial-analysis` 返回的全部近30年数据
- ROCE：使用 `roce-calculator` 计算的实际值
- 技术指标：使用 `technical-analyzer` 返回的实际值
- 如果某年份数据缺失（如ROCE），显示为"-"，不要填写
---

## ✅ 完整工作流 Todo 清单

> 按顺序执行，每完成一步勾选一项。

### ▷ 阶段一：数据采集（调用分析 Skills）

- [ ] `stock-analyzer` — 获取基本信息+估值（PE/PB/股息率/行业）
- [ ] `financial-analysis` — 获取财务数据（近30年营收/净利润/ROE/负债率）
- [ ] `shareholder-deep` — 获取股东结构（十大股东/增减持）
- [ ] `technical-analyzer` — 获取技术指标（RSI/MA均线/趋势）
- [ ] `market-analyzer` — 获取市场环境（平均PE/牛熊判断）
- [ ] `roce-calculator` — 获取ROCE趋势（近10年实际计算值）
- [ ] `a-dividend-analyzer` — 获取分红历史（连续分红年数/股息率）
- [ ] `risk-analysis` — 获取风险+分位数（新闻风险/价格分位）

### ▷ 阶段二：报告组装

- [ ] 按 `guides/unified-template.md` 模板填充10章报告
- [ ] 所有数值使用 Skill 返回的原始值，不修改不遗漏
- [ ] 财务表包含 `financial-analysis` 返回的全部年份
- [ ] ROCE 取 `roce-calculator` 的实际计算值（非财务分析近似值）
- [ ] 缺失数据显示为"-"，不填充假设数据
- [ ] 6维度综合评分计算正确，总分在0-120范围内

### ▷ 阶段三：质量复核

- [ ] 按 `guides/data-objectivity-review-guide.md` 逐项审核数据准确性
- [ ] 每个观点/结论都有数据支撑
- [ ] 表述中立，无主观臆断
- [ ] 投资建议与评分逻辑自洽
- [ ] 风险提示充分，无遗漏

### ▷ 阶段四：交付

- [ ] 报告文件生成到 `output/` 目录
- [ ] 文件名格式：`{代码}_{名称}_综合分析报告_{日期}.md`
- [ ] 包含免责声明、数据截止日期、数据来源列表
- [ ] 如有需要，通过 `email-sender` 发送报告附件

## 使用方式

**本 Skill 为报告生成指南。严格依次调用以下 Skills 获取数据后，你组装生成报告**

### 模板文件

统一模板位于：`./guides/unified-template.md`

包含：10章报告结构 + 8个数据源Skills映射 + 报告生成流程

### 步骤1：调用分析 Skills（在 OpenCode 中执行）

```
# 在对话框中依次调用这些 Skills 获取数据
1. stock-analyzer    → 基本信息+估值   skill(name="stock-analyzer")
    → {"pe": 7.7, "pb": 1.47, "股息率": 7.5%}

2. financial-analysis → 财务数据（近30年全部）
   skill(name="financial-analysis")
   → 返回29年数据：1997-2025，包含营收/净利润/ROE/ROCE/负债率

3. shareholder-deep → 股东结构
   skill(name="shareholder-deep")
   → {"集中度": 37.9%, "减持": 5次}

4. technical-analyzer → 技术指标   skill(name="technical-analyzer")
   → {"rsi": 78.24, "趋势": "震荡"}

5. market-analyzer → 市场环境
   skill(name="market-analyzer")
   → {"pe": 34.25, "趋势": "牛市"}

6. roce-calculator → ROCE趋势（实际计算值，不是财务分析中的近似值）
   skill(name="roce-calculator")
   → {"2025": 19.12%, "2024": 20.1%, ...}

7. a-dividend-analyzer → 分红历史
   skill(name="a-dividend-analyzer")
   → {"连续": 65年}

8. risk-analysis → 风险+分位数   skill(name="risk-analysis")
   → {"分位数": 94.4%}
```

**严格按 Skills 返回的数据生成报告，不要修改**
- 财务表必须包含 `financial-analysis` 返回的全部年份（近10年）
- ROCE 取 `roce-calculator` 返回的实际计算值
- 技术指标取 `technical-analyzer` 返回的实际值
- 如果某数据缺失，显示"-"，不要填写
### 步骤2：组装报告

**严格按 Skills 返回的原始数据生成报告，不要修改、遗漏或填充假设值。**

收集上述 Skills 返回的数据后，你根据模板组装生成 Markdown 报告，包含：
- 10章标准结构
- 6维度综合评分
- 投资建议

### 步骤3：质量复核（必需）

**报告生成后必须按照 `guides/data-objectivity-review-guide.md` 逐项审核数据准确性和观点客观性。**

- 参照 `guides/data-objectivity-review-guide.md` 检查数据转录是否准确
- 检查每个观点是否有数据支撑
- 确认投资建议与评分逻辑自洽
- 审核通过后方可交付

## ⭐ 综合评分：6维度评分卡（满分120分，必须执行）

> 评分卡定义了每个维度的评分规则。你必须从各技能返回的原始数据中提取数值，按以下规则逐维度打分。

### 评分卡

| # | 维度 | 满分 | 数据来源（技能 + 字段） | 评分规则 |
|:-:|:----|:---:|:---------------------|:---------|
| 1 | **盈利能力** | 20 | ROCE `roce-calculator` | ROCE > 25% → 20分；20-25% → 16分；15-20% → 10分；10-15% → 5分；<10% → 0分；ROCE趋势下降（近3年<早3年均×0.9）→ 扣5分 |
| 2 | **财务安全** | 20 | 资产负债率 `financial-analysis` + 流动比率 `financial-analysis` | 基础分10分。流动比率>2.0 +5分，1.0-2.0 +2分，<1.0 -3分；负债率<40% +5分，40-60% +2分，60-75% -2分，>75% -5分 |
| 3 | **估值合理性** | 20 | PE(TTM) `stock-analyzer` + PB `stock-analyzer` | PE<10 → 10分，10-15 → 8分，15-25 → 5分，25-40 → 3分，>40 → 0分；+ PB<1.5 → 10分，1.5-3 → 7分，3-5 → 3分，>5 → 1分 |
| 4 | **技术面** | 20 | MA信号 `technical-analyzer` + RSI `technical-analyzer` | 基础分10分。金叉(MA50>MA200) +5分，死叉 -5分；RSI<30(超卖) +3分，RSI>70(超买) -3分；RSI 30-70正常 +0分 |
| 5 | **业务前景** | 20 | 行业地位 `stock-analyzer` + 分位数 `percentile-analyzer` | 基础分10分。行业龙头（基于你判断） +5分；价格分位<30%(低估区间) +5分，30-50% +2分，50-70% +0分，>70% -3分 |
| 6 | **新闻风险** | 20 | 新闻分析 `risk-analysis` | 诚信风险 → 5分；经营风险(高) → 8分；经营风险(中) → 12分；全部低风险 → 20分；无新闻数据 → 10分(保守) |

### 综合判定

| 总分 | 评级 | 建议 |
|:---:|:----:|:-----|
| **100-120分** | 🟢 **A级-优秀** | 积极关注，可考虑建仓 |
| **80-99分** | 🟢 **B级-良好** | 基本面良好，逢低布局 |
| **60-79分** | 🟡 **C级-一般** | 观望为主，等待更好时机 |
| **40-59分** | 🟠 **D级-较差** | 风险较高，谨慎对待 |
| **0-39分** | 🔴 **E级-危险** | 回避，风险过大 |

### 评分示例

```
输入（格力电器 000651）：
  维度1 盈利能力: ROCE 22.57%（20-25%）→ 16分
    ROCE趋势下降 → -5分, 实得11分
  维度2 财务安全: 负债率56.83%（40-60%）→ +2分
    流动比率1.12（1.0-2.0）→ +2分
    基础10+2+2=14分
  维度3 估值: PE 7.46（<10）→ 10分 + PB 1.43（<1.5）→ 10分 = 20分
  维度4 技术面: 基础10分, 死叉 -5分, RSI 54.29正常 +0分 = 5分
  维度5 业务前景: 基础10分, 空调龙头 +5分, 分位20.7%（<30%）+5分 = 20分
  维度6 新闻风险: 全部低风险 → 20分
  ————————————————————————
  总分: 11+14+20+5+20+20 = 90/120 → 🟢 B级良好
  建议: 基本面良好，逢低布局
```
