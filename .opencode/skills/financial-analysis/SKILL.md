## 输出维度（步骤）

### Step 1: 环境初始化
- 股票代码、名称、行业、上市日期
- PE / PB / 总市值

### Step 2: 数据获取
- 利润表（近5年）
- 资产负债表（近5年）
- 现金流量表（近5年）
- 财务指标（近5年）

### Step 3: 核心指标计算
| 指标 | 公式 | 用途 |
|------|------|------|
| ROE | 净利润 / 股东权益 | 盈利能力 |
| 毛利率 | (营收-成本)/营收 | 盈利质量 |
| 净利润率 | 净利润 / 营收 | 盈利效率 |
| 资产负债率 | 总负债 / 总资产 | 偿债风险 |
| 营收增长 | 同比增长率 | 成长性 |
| 自由现金流 | 经营现金流 - 资本支出 | 现金健康度 |

### Step 4: 风险筛查
| 风险类型 | 触发条件 | 信号 |
|----------|----------|:----:|
| 盈利质量 | 经营现金流 < 净利润×0.7 | ⚠️ 预警 |
| 存货积压 | 存货增长 > 营收增长 | ⚠️ 预警 |
| 商誉减值 | 商誉/总资产 > 20% | ⚠️ 预警 |
| 存贷双高 | 货币资金占比高 + 有息负债高 | ⚠️ 预警 |

## ⭐ 你的评估任务：6维度评分卡 + 综合健康判定（必须执行）

> main.py 只输出原始数据。你必须基于以下规则，从 stock-analyzer + roce-calculator 的原始数据中自行提取数值，逐项评分。

|   |      |         | 15-20% | 10 |
|   |      |         | 10-15% | 5 |
|   |      |         | < 10% | 0 |
| A2 | **毛利率**（取最新财年） | 利润表 `毛利率` | > 40% | 10 |
|   |      |         | 25-40% | 7 |
|   |      |         | 15-25% | 4 |
|   |      |         | < 15% | 1 |
| A3 | **净利润率**（取最新财年） | 利润表 `净利润率` | > 15% | 5 |
|   |      |         | 8-15% | 3 |
|   |      |         | < 8% | 1 |

### 维度B：ROCE趋势（满分20分，权重20%）

| # | 指标 | 数据来源 | 判定条件 | 得分 |
|---|------|---------|---------|:----:|
| B1 | **ROCE绝对值** | roce-calculator 输出 | > 25% | 10 |
|   |      |         | 20-25% | 8 |
|   |      |         | 15-20% | 5 |
|   |      |         | 10-15% | 3 |
|   |      |         | < 10% | 0 |
| B2 | **ROCE趋势** | 比较 `近3年均值 vs 早3年均值` | 近3年 ≥ 早3年均值×0.9（上升或平稳） | 10 |
|   |      |         | 近3年 < 早3年均值×0.9（下降） | 0 |
|   |      |         | 近3年 < 早3年均值×0.5（严重恶化） | -5 |

### 维度C：偿债能力（满分20分，权重20%）

| # | 指标 | 数据来源 | 判定条件 | 得分 |
|---|------|---------|---------|:----:|
| C1 | **资产负债率** | 财务数据 `资产负债率` | < 40%（非常安全） | 10 |
|   |      |         | 40-60%（安全，制造业可宽限到<70%） | 7 |
|   |      |         | 60-75%（警戒，需关注） | 4 |
|   |      |         | > 75%（危险） | 0 |
| C2 | **流动比率** | 财务数据 `流动比率` | > 2.0（优秀） | 10 |
|   |      |         | 1.5-2.0（良好） | 7 |
|   |      |         | 1.0-1.5（一般） | 4 |
|   |      |         | < 1.0（短期偿债压力） | 0 |

### 维度D：现金流质量（满分15分，权重15%）

| # | 指标 | 数据来源 | 判定条件 | 得分 |
|---|------|---------|---------|:----:|
| D1 | **经营现金流/净利润** | 现金流量表 + 利润表 | ≥ 1.0（盈利质量极好） | 10 |
|   |      |         | 0.7-1.0（盈利质量正常） | 7 |
|   |      |         | 0.5-0.7（含水分，关注） | 3 |
|   |      |         | < 0.5（盈利质量差） | 0 |
| D2 | **自由现金流** | 经营现金流 - 资本支出 | 正值且持续增长 | 5 |
|   |      |         | 正值但波动 | 3 |
|   |      |         | 负值 | 0 |

### 维度E：成长性（满分10分，权重10%）

| # | 指标 | 数据来源 | 判定条件 | 得分 |
|---|------|---------|---------|:----:|
| E1 | **营收增长率**（最新财年） | 利润表 `营收同比` | > 15%（高速增长） | 5 |
|   |      |         | 5-15%（稳健增长） | 3 |
|   |      |         | 0-5%（低速增长） | 1 |
|   |      |         | < 0%（下滑） | 0 |
| E2 | **净利润增长率**（最新财年） | 利润表 `净利润同比` | > 15%（高速增长） | 5 |
|   |      |         | 5-15%（稳健增长） | 3 |
|   |      |         | 0-5%（低速增长） | 1 |
|   |      |         | < 0%（下滑） | 0 |

### 维度F：风险扣分（满分-5分，扣分项）

| # | 风险项 | 数据来源 | 触发条件 | 扣分 |
|---|--------|---------|---------|:----:|
| F1 | **盈利预警** | 风险筛查结果 | 经营现金流 < 净利润×0.7 | -2 |
| F2 | **存货预警** | 风险筛查结果 | 存货增长 > 营收增长 | -1 |
| F3 | **商誉预警** | 风险筛查结果 | 商誉/总资产 > 20% | -1 |
| F4 | **ROCE严重恶化** | ROCE趋势数据 | 近3年均值 < 早3年均×0.5 | -1 |

---

### 综合评分计算

```公式
总分 = A1+A2+A3 + B1+B2 + C1+C2 + D1+D2 + E1+E2 + F1+F2+F3+F4
满分 = 15+10+5 + 10+10 + 10+10 + 10+5 + 5+5 + 0 = 95分（不含扣分）
```

| 总分区间 | 综合评级 | 含义 |
|:--------:|:--------:|:----|
| **80-95分** | 🟢 **A级-优秀** | 财务非常健康，基本面强劲 |
| **65-79分** | 🟢 **B级-良好** | 财务健康，个别指标需关注 |
| **50-64分** | 🟡 **C级-一般** | 财务状况中等，需进一步调研 |
| **35-49分** | 🟠 **D级-较差** | 多项指标亮红灯，高度谨慎 |
| **0-34分** | 🔴 **E级-危险** | 财务恶化，建议回避 |

### 一票否决条件（触发任何一项，综合评级直接降一级或两级）

| 条件 | 降级规则 |
|:-----|:---------|
| 资产负债率 > 75% | 综合评级降一级 |
| 经营现金流/净利润 < 0.3 | 综合评级降两级 |
| 营收连续2年下降 > 20% | 综合评级降一级 |
| ROE连续3年 < 5% | 综合评级降为 E 级 |

### 评分示例

```
输入数据（格力电器 000651）：
  ROE: 19.78% → A1得10分
  毛利率: 26.5%（假设值）→ A2得7分
  净利润率: 13.5% → A3得3分
  ROCE近5年均值: 22.57% → B1得8分
  ROCE趋势: 下降(22.62% < 34.83%×0.9) → B2得0分
  资产负债率: 56.83% → C1得7分
  流动比率: 1.12 → C2得4分
  OCF/净利润: 1.61x → D1得10分
  自由现金流: 正值 → D2得5分
  营收增长: 约0%（平稳）→ E1得1分
  净利润增长: 约4.5% → E2得1分
  风险扣分: 无触发 → F扣0分
  ————————————————————————
  总分: 10+7+3 + 8+0 + 7+4 + 10+5 + 1+1 + 0 = 56分 → 🟡 C级一般
  （受营收增速低和ROCE趋势下降影响较大）
```

