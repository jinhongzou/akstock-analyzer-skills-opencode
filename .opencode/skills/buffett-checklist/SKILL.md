# 巴菲特问题清单检查（Buffett Checklist）

**功能**: 对A股个股运行巴菲特7关检查清单，你基于原始数据独立评估每关（含市场风险判断）  
**数据源**: 聚合 Tushare Pro + 新浪财经 + 东方财富（通过现有 analysers）

---

## 使用方式

**OpenCode 调用**:
```
skill(name="buffett-checklist")
```

**命令行运行**:
```bash
python .opencode/skills/core/src/skills/buffett-checklist/main.py <股票代码>
```

### 示例

```bash
python .opencode/skills/core/src/skills/buffett-checklist/main.py 600519   # 贵州茅台
python .opencode/skills/core/src/skills/buffett-checklist/main.py 000651   # 格力电器
python .opencode/skills/core/src/skills/buffett-checklist/main.py 002594   # 比亚迪
```

## 你评估指引（必须阅读）

> ⚠️ **本技能只输出原始聚合数据，不包含你评估结论。**
> 你在生成报告时必须自己对数据进行分析评估，不得直接复述main.py输出的数值。

### 执行流程

```
Step 1: 运行 main.py <股票代码>         → 获取7类个股原始数据
Step 2: 运行 market-analyzer/main.py    → 获取市场整体估值/均线趋势（可选数据源1）
Step 3: 运行 market-systemic-risk/main.py → 获取宏观经济/PE分位/资金面（可选数据源2）
Step 4: 对每关逐条核对检查项            → 对照下方的"决策规则"逐条判定
Step 5: 汇总7关结果                     → 生成综合总览表
Step 6: 镜子测试 + 关键风险 + 监控指标  → 补充定性分析
Step 7: 输出完整报告                    → 按最终输出格式要求排版
```

> **注意**: Step 2 和 Step 3 为第七关（市场风险）的专用数据源。如果不运行这两个脚本，第七关的"宏观数据"部分将依靠你自身知识判断，并在报告中标注"未运行市场分析脚本，基于你自身认知评估"。

### 核心原则

| 原则 | 说明 |
|------|------|
| **每关独立评分** | 7关互不干扰，不因某一关优秀而放松另一关标准 |
| **数据说话** | 每个结论必须有 main.py 中的数据行作为证据支撑 |
| **诚实标注不确定性** | 不确定时必须写"你无法判断(原因)"，不得伪造认知 |
| **商业常识补充** | 数据之外，你应结合对行业/公司的认知做定性判断 |

## 七关检查框架（含详细决策规则）

---

### 第一关：能力圈 — 我能理解这门生意吗？

#### 检查项与决策规则

每条检查项按 **2分制** 打分：通过=2，模糊=1，不通过=0。总分 ≥5 为✅在圈内，3-4为⚠️边界，≤2为❌在圈外。

| # | 检查项 | 决策规则（你必须逐条执行） | 数据来源 |
|---|--------|--------------------------|---------|
| 1 | **能用一句话说清这个公司怎么赚钱吗？** | 自问自答，写出1句话。如果能写出来 → 通过(2分)；只能写出模糊概念（如"做东西卖的"）→ 模糊(1分)；完全写不出 → 不通过(0分) | main.py "行业" 字段 + 你认知 |
| 2 | **它的客户是谁？为什么客户选它而不选竞争对手？** | 列出客户类型（B端/C端/政府）+ 客户选择它的理由。能说清 → 通过(2分)；只能说清客户类型但说不出为什么 → 模糊(1分)；说不出 → 不通过(0分) | 你认知（数据无法直接给出） |
| 3 | **未来10年这门生意大概率还在做同样的事吗？** | 判断行业可预测性。消费品/公用事业/银行 → 通过(2分)；科技/医药有不确定性 → 模糊(1分)；周期/新兴/政策敏感 → 不通过(0分) | main.py "行业" 字段 |
| 4 | **决定这个公司成败的3个关键变量是什么？** | 列出3个具体变量。能写出具体变量 → 通过(2分)；只能写出1-2个 → 模糊(1分)；写不出 → 不通过(0分) | 你认知 |

#### 最终判定

| 总分 | 结果 | 含义 |
|------|------|------|
| 6-8 分 | ✅ 在圈内 | 对业务有清晰理解，可以进入后续关卡 |
| 3-5 分 | ⚠️ 边界区域 | 理解不完整，需要更多研究或外部信息才能决策 |
| 0-2 分 | ❌ 在圈外 | 不理解这门生意，巴菲特规则：**直接放弃，不进入后续关卡** |

> **注意**：如果第一关判定为❌在圈外，仍需输出报告但需明确标注"第一关未通过，不建议投资"。

### 第二关：护城河 — 有持久的竞争优势吗？

#### 评分卡（5项，每项最高3★，总计15★）

| # | 指标 | 数据来源（main.py具体行） | 判定条件 | ★ |
|---|------|-------------------------|---------|----|
| 1 | **ROCE近5年均值** | `近5年ROCE均值: XX%` | >25% 卓越 | ★★★ |
|   |      |                         | 20-25% 优秀 | ★★ |
|   |      |                         | 15-20% 良好 | ★ |
|   |      |                         | <15% 不足 | 0 |
| 2 | **ROCE趋势** | `趋势方向=上升/下降/平稳` | 近3年 ≥ 早3年均值（上升或平稳） | ★★ |
|   |      |                         | 近3年 < 早3年均值×0.9（下降） | 0 |
| 3 | **连续现金分红年数** | `现金分红年份数: XX` | ≥20年 | ★★★ |
|   |      |                         | 10-19年 | ★★ |
|   |      |                         | 5-9年 | ★ |
|   |      |                         | <5年 | 0 |
| 4 | **行业地位** | 你基于"行业"判断 | 公认龙头/寡头（如茅台在白酒） | ★★★ |
|   |      |                         | 行业前5/有特色 | ★★ |
|   |      |                         | 中小型/竞争激烈 | ★ |
|   |      |                         | 边缘参与者 | 0 |
| 5 | **定价权判断** | 你结合行业+ROCE综合判断 | 能提价且客户不流失（奢侈品/品牌消费品/必需品） | ★★★ |
|   |      |                         | 有一定议价能力但有限 | ★★ |
|   |      |                         | 价格接受者/完全竞争 | ★ |
|   |      |                         | 无定价权/靠降价竞争 | 0 |

#### 最终判定

| 总★ | 结果 | 含义 |
|-----|------|------|
| ≥12★ | 🌟🌟🌟 宽护城河 | 竞争优势强大，可持续 |
| 8-11★ | 🌟🌟 中等护城河 | 有优势但非不可逾越 |
| 4-7★ | 🌟 窄护城河 | 竞争优势有限 |
| ≤3★ | ❌ 无护城河 | 不具备长期竞争优势 |

### 第三关：管理层 — 管理层值得信任吗？

#### 检查项与决策规则（6项检查，按红灯累计判定）

| # | 检查项 | 数据来源（main.py具体行） | 判定逻辑 | 结果影响 |
|---|--------|-------------------------|---------|---------|
| 1 | **大股东结构** | `#1 XXX: ...(XX%)` → 看第一大股东名称和比例 | 国资/知名机构/创始人 → 信任加分；不明背景的壳公司 → 扣分 | +1 / -1 |
|   |          | `十大股东合计持股: XX%` | >50% 股权集中(稳定) → 加分；<30% 过于分散 → 关注 |   |
| 2 | **股权质押** | `总质押比例: XX%` | 无质押 → +1；<20% → 0；≥20% 或 "高" → -2（红灯） | +1/0/-2 |
| 3 | **股份回购** | `回购次数: X` | ≥3次 → +1；1-2次 → 0；无 → 0 | +1/0 |
| 4 | **分红连续性** | `分红连续性评价: XXX` | "优秀" → +1；"良好" → 0；其他 → -1 | +1/0/-1 |
| 5 | **融资历史** | 你自行判断（需搜索了解） | 从未增发/少量增发 → +1；频繁增发/配股 → -2 | +1/0/-2 |
| 6 | **诚信记录** | 你自行判断（需搜索了解） | 无重大违规/财务造假记录 → 0；有记录 → **直接❌不信任，放弃** | 0/一票否决 |

#### 最终判定

| 红灯合计 | 结果 | 含义 |
|---------|------|------|
| 一票否决（诚信问题） | ❌ 不信任 | 管理层不可信，直接放弃 |
| 总扣分 ≥ -3 | ❌ 不信任 | 多个红灯，风险过高 |
| -2 到 +1 | ⚠️ 关注 | 需要进一步验证，降低仓位 |
| ≥ +2 | ✅ 优秀 | 管理层可靠，与股东利益一致 |

### 第四关：财务健康 — 财务指标是否优秀？

#### 检查项与判定规则（5项，按"通过数"判定）

| # | 指标 | 数据来源（main.py具体行） | 通过条件 | 备注 |
|---|------|-------------------------|---------|------|
| 1 | **ROCE近3年均值** | `近3年ROCE均值: XX%` | ≥15% 通过 | 巴菲特底线标准 |
| 2 | **资产负债率** | `最新资产负债率: XX%` | <60% 通过 | 格力等制造业可放宽到<70% |
| 3 | **净现金** | `净现金: +XX亿` | >0（正值）通过 | 负值意味着净负债 |
| 4 | **经营现金流质量** | `XXXX OCF / XXXX年净利润: XXx` | ≥0.7x 通过 | 低于0.7x说明盈利含"水分" |
|   |                 | （若无此数据行，手动计算：经营现金流÷净利润） |           |     |
| 5 | **ROE** | `最新ROE: XX%` | ≥15% 通过 | 巴菲特标准 |

#### 最终判定

| 通过项数 | 结果 | 含义 |
|---------|------|------|
| 5/5 通过 | ✅ 健康 | 财务非常健康 |
| 3-4/5 通过 | ⚠️ 关注 | 个别指标不达标，需注意 |
| ≤2/5 通过 | ❌ 危险 | 多项财务指标亮红灯 |

> **注意**：ROCE < 10% 或 资产负债率 > 75% 时，即使其他项通过也应降级为⚠️关注

### 第五关：安全边际 — 价格足够便宜吗？

#### 分项判定（5项独立判定，按"充足数"汇总）

| # | 指标 | 数据来源（main.py具体行） | 充足条件 | 一般条件 | 不足条件 |
|---|------|-------------------------|---------|---------|---------|
| 1 | **PE(TTM)** | `市盈率(TTM): XX` | <10 | 10-15 | >15 |
| 2 | **PB** | `市净率: XX` | <1.5 | 1.5-3.0 | >3.0 |
| 3 | **股息率(TTM)** | `股息率(TTM): X%` | >5% | 3-5% | <3% |
| 4 | **52周价格分位** | `当前位置分位: X%` | <25%（底部区域） | 25-50%（偏低） | >50%（中部以上） |
| 5 | **股息率 vs 国债** | 你自行查询当前中国10年期国债收益率 | 股息率 > 国债×1.5 | 股息率 > 国债 | 股息率 ≤ 国债 |

#### 综合判定

| 充足项数 | 结果 | 含义 |
|---------|------|------|
| ≥3项充足 + 无不足 | ✅ 充足 | 价格便宜，安全边际大 |
| 2-3项充足但有不足 | ⚠️ 一般 | 有一定安全边际但不够充分 |
| ≤1项充足 | ❌ 不足 | 价格不便宜，缺乏安全边际 |

### 第六关：红旗检查 — 绝不触碰的7条红线

#### 逐条检测（每条独立标记 ✅未触发 或 ⚠️触发）

| # | 红线 | 检测方法（精确到main.py输出行） | 触发条件 | 备注 |
|---|------|-------------------------------|---------|------|
| 1 | **护城河依赖关键人物** | 你判断：该行业/公司是否极度依赖某个个人 | ① 创始人兼CEO持股极高且无接班人计划；② 品牌与个人强绑定（如格力/董明珠、小米/雷军） → 触发 | 如果是消费品品牌公司（如茅台），品牌力＞个人影响力，不触发 |
| 2 | **资产负债率>70%** | 从 `最新资产负债率: XX%` 读取 | >70% → 触发 | 金融/保险行业除外（高杠杆天然属性） |
| 3 | **经营现金流/净利润<0.7** | 从 `XXXX OCF / XXXX年净利润: XXx` 读取 | <0.7x → 触发 | 用最新完整年度的数据判断 |
| 4 | **ROCE连续3年下降>50%** | 从ROCE表格手动计算：`(3年前ROCE - 最新ROCE) / 3年前ROCE` | 下降幅度 > 50% → 触发 | 如2023年40.89%→2025年28.58%，降幅30%→不触发 |
| 5 | **大股东高比例质押** | 从 `总质押比例: XX%` 读取 | ≥30% → 触发 | 30%意味着大股东面临平仓风险 |
| 6 | **纯商品/无差异化行业** | 你基于"行业"判断 | 行业特征是纯商品（钢铁/煤炭/化工/航运/养殖/芯片代工…）且公司无明显差异化 → 触发 | 品牌消费品（白酒/家电/调味品）不属于纯商品 |
| 7 | **频繁增发或复杂关联交易** | 你自行检查（需搜索或已知信息） | ① 近5年增发≥2次且增发后ROE下降；② 有大量关联交易且利益输送嫌疑 → 触发 | 需你基于自身知识判断，不确定时标注"你无法判断" |

#### 最终判定

| 触发数 | 结果 |
|--------|------|
| 0条 | ✅ 无红旗 |
| 1-2条 | ⚠️ 注意（需评估严重性，可能降级） |
| ≥3条 | ❌ 否决（建议放弃） |

> 触发红条款数超3条时，即使其他5关都通过，也应建议"放弃"或"极为谨慎"。

### 第七关：当前市场风险 — 市场整体环境是否适合买入？

> ⚠️ **第七关评估的是市场系统性风险，与前面6关（评估公司自身质量）相互独立。**
> 好公司也可能因市场整体高估而成为糟糕的投资。巴菲特说"别人贪婪时恐惧"，市场过热时即使公司优秀也应谨慎。

#### 数据来源说明

第七关需要额外的市场数据。获取方式（二选一）：

| 方式 | 命令 | 获取的数据 |
|------|------|-----------|
| **推荐** | `python .opencode/skills/core/src/skills/market-systemic-risk/main.py` | 宏观经济(GDP/CPI/PMI/M2)、PE/PB分位、资金流向、历史对比 |
| **快速** | `python .opencode/skills/core/src/skills/market-analyzer/main.py` | 全市场平均PE、上证MA20/MA50、牛熊判断 |

> 如果未运行以上脚本，你应基于自身对当前市场环境的认知做定性判断，并在报告中明确标注"未获取实时市场数据，基于你认知评估"。

#### 评分卡（5项，每项最高4分，总计20分）

| # | 检查项 | 数据来源（精确到main.py输出行） | 4分(安全) | 2-3分(中性) | 0-1分(危险) | 你判断要点 |
|---|--------|-------------------------------|----------|------------|------------|-----------|
| 1 | **全市场PE估值分位** | `PE分位(5年): XX%`（market-systemic-risk） | < 30%（低位区间） | 30-70%（中位合理） | > 70%（高位区间） | PE分位越高说明市场越贵，未来回报预期越低 |
| 2 | **技术面趋势（牛熊判断）** | `市场状态: XX`（market-analyzer）或 `MA20: XX / MA50: XX`（market-systemic-risk） | MA20 > MA50 且 60日涨幅温和(<15%) | MA20 ≈ MA50 横盘震荡 | MA20 < MA50（熊市）或 短期暴涨(60日涨幅>20%) | 牛市初期安全，牛市末期(暴涨后)危险 |
| 3 | **宏观经济健康度** | `GDP同比: X%`, `PMI制造业: X`, `M2同比: X%`（market-systemic-risk） | GDP>5% + PMI>50 + M2适度 | 2项达标 | 仅1项达标或全部恶化 | 经济基本面决定企业盈利大环境 |
| 4 | **市场情绪与资金面** | `上证融资余额: X亿`, `量比: X`（market-systemic-risk） | 融资余额温和 且 量比<1.2（正常） | 融资余额偏高或量比1.2-1.5 | 融资余额极端高或量比>1.5（过热）或融资余额持续下降（恐慌） | 融资余额反映杠杆情绪，过高意味贪婪 |
| 5 | **当前PE vs 历史极端值** | `2015年股灾PE峰值: X`, `2018年PE谷值: X`, `近期PE区间: [min,max]`（market-systemic-risk） | 当前PE < 2018年谷值×1.5（偏低） | 当前PE在2018谷值~2015峰值之间 | 当前PE > 2015年峰值（接近过热） | 对比历史极端值判断当前位置 |

#### AI打分指南

```
第1项（PE分位）：直接从 market-systemic-risk 输出读取，客观数值
第2项（技术面）：从 market-analyzer 或 market-systemic-risk 读取均线数据
第3项（宏观）：从 market-systemic-risk 读取，你需综合判断趋势方向
第4项（情绪）：从 market-systemic-risk 读取，需结合你的市场感知
第5项（历史对比）：从 market-systemic-risk 读取，纯客观对比
```

> 如果某个数据行不存在（如未运行对应脚本），该项评为1分并在报告中注明"数据缺失，保守评分"。

#### 最终判定

| 总分(满分20) | 结果 | 含义 |
|-------------|------|------|
| 16-20分 | 🟢 **市场友好** | 市场估值合理偏低，宏观健康，适合建仓 |
| 11-15分 | 🟡 **市场中性** | 市场无明显过热或过冷，正常配置 |
| 6-10分 | 🟠 **市场承压** | 市场估值偏高或宏观走弱，降低预期仓位 |
| 0-5分 | 🔴 **市场危险** | 市场明显过热或系统性风险高，**建议大幅减仓或观望** |

#### 第七关与总体结论的关系

第七关不参与传统"通过/不通过"计数（因为它评估的是市场环境而非公司质量），而是作为**总体建议的加权调节器**：

| 前6关通过数 + 第七关结果 | 总体建议 |
|------------------------|---------|
| 前6关≥4关通过 + 🟢市场友好 | **强烈推荐买入** |
| 前6关≥4关通过 + 🟡市场中性 | **推荐买入，正常仓位** |
| 前6关≥4关通过 + 🟠市场承压 | **可以买入，但降低仓位** |
| 前6关≥4关通过 + 🔴市场危险 | **等待更好时机，即使公司优秀** |
| 前6关<4关通过 | 无论市场环境如何，**不推荐买入** |

## 设计原则

1. **你独立评估**：main.py 只做数据聚合，所有"通过/不通过/关注"的判断由 你基于数据和商业认知独立完成
2. **决策规则驱动**：每关的评估标准都已结构化，你必须逐条对照执行，不得跳过任何检查项
3. **诚实展示不确定性**：你对不确定的领域应明确标注"你无法判断(原因)"，不得伪造认知
4. **数据痕迹可追溯**：每个结论必须注明数据来源行（如"从 `最新资产负债率: 56.83%` 读取"）
5. **公司质量与市场环境分离**：前6关评估公司自身质量，第7关评估市场系统性风险，两者独立评分后再综合决策
