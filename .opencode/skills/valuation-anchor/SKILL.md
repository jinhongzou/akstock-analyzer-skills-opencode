# Valuation Anchor Method

**功能**: 估值锚点法 - 多维度交叉验证的买入点定价框架  
**数据源**: akshare + 内部计算逻辑

---

## 使用方式

**命令行运行**:
```bash
python .opencode/skills/core/src/skills/valuation-anchor/main.py <股票代码>
```

### 示例

```bash

```

## 方法概述

买入点 = 多个独立估值维度交叉验证后的价格区间。单一维度不够可靠，需要至少3个维度同时指向同一区间。

---

## 维度1：52周价格分布法

**原理**: 一年内的价格分布反映了市场对该股的全部定价共识。在中位数以下买入，统计上赢面更大。

**步骤**:
1. 获取 52周最高价（H）和最低价（L）
2. 计算中位数 M = (H + L) / 2
3. 计算 1/4 分位 Q1 = L + (H - L) * 0.25
4. 计算 3/4 分位 Q3 = L + (H - L) * 0.75

**定价规则**:

| 策略 | 价格区间 | 含义 |
|------|---------|------|
| 激进 | Q1 ~ M | 回调到中位数以下 |
| 稳健 | L + (M - L) * 0.2 ~ Q1 | 偏低1/4区间 |
| 理想 | L 附近 | 极端低价 |

---

## 维度2：PE 合理估值法

**原理**: 用当前每股收益（EPS）乘以合理PE区间，得到合理价格范围。

**步骤**:
1. 获取当前 EPS（最近12个月或最新年报）
2. 确定该行业的合理PE区间（参考同类公司PE均值和历史PE中位数）
3. 合理价格 = EPS * PE合理区间

**常见行业PE参考**:

| 行业类型 | 低估PE | 合理PE | 高估PE |
|---------|--------|--------|--------|
| 银行 | <5 | 5-7 | >8 |
| 公用事业 | <8 | 8-12 | >14 |
| 消费品（成熟） | <15 | 15-20 | >25 |
| 消费品（成长） | <20 | 20-35 | >40 |
| 科技/医药 | <25 | 25-40 | >50 |
| 周期股 | 看PB | — | — |

**定价规则**:

| 策略 | 价格区间 | 含义 |
|------|---------|------|
| 激进 | EPS * 合理PE下限 | 回到合理偏下 |
| 稳健 | EPS * (低估PE ~ 合理PE下限) | 明显偏低 |
| 理想 | EPS * 低估PE | 极端低估 |

---

## 维度3：股息率锚定法

**原理**: 对高分红股票，股息率是最硬的"确定性收益率"锚点。用目标股息率倒推目标买入价。

**步骤**:
1. 获取最近年度每股分红（DPS）
2. 设定目标股息率门槛：
   - 基准门槛：4%（跑赢银行理财）
   - 优秀门槛：5%（有吸引力的分红收益）
   - 极致门槛：6%（难得的底部机会）
3. 目标价格 = DPS / 目标股息率

**定价规则**:

| 策略 | 目标股息率 | 含义 |
|------|-----------|------|
| 激进 | >=4% | 略有分红优势 |
| 稳健 | >=5% | 明显的分红吸引力 |
| 理想 | >=6% | 难得的底部机会 |

**适用条件**: 仅适用于分红率稳定（>50%）、分红连续（>3年）的公司。

---

## 维度4：技术面验证

**原理**: 基本面和估值算出的价格区间，需要技术面验证是否实际有支撑。

**关键指标**:
- **历史支撑位**: 过去一年多次回踩后反弹的价格区域
- **均线支撑**: 60日/120日均线位置（长期均线支撑更强）
- **成交密集区**: 长期横盘整理的价格区间
- **缺口/前高**: 重要价格关口

**验证规则**:
- 估值算出的买入区间内有技术面支撑 → ✅ 增强信心
- 估值算出的买入区间内无技术面支撑 → ⚠️ 降低仓位或观望
- 技术面支撑位远高于估值买入区间 → 重新评估估值假设

---

## ⭐ 交叉验证与最终定价 — 可执行决策规则

### 维度权重分配（根据股票类型自动选择）

| 股票类型 | 判定条件 | 维度1 价格分布 | 维度2 PE估值 | 维度3 股息率 | 维度4 技术面 |
|---------|---------|:-------------:|:-----------:|:-----------:|:-----------:|
| **高分红价值型** | 股息率 > 4% 且 分红连续>5年 | 25% | 25% | **35%** ⭐ | 15% |
| **成长型** | 营收增速 > 15% 或 PE > 40 | 30% | 20% | 不适用 | 20% |
| **周期型** | 行业=银行/钢铁/煤炭/有色/养殖等 | 30% | 15% | 25% | 30% |
| **普通消费品** | 以上都不符合 | 25% | 30% | 25% | 20% |
| **亏损/微利** | EPS < 0 或 PE为负 | 40% | 不适用 | 不适用 | 60% |

> **规则**：先判断股票类型，再按对应权重计算加权目标价。

### 维度对齐度与置信度

对4个维度各自算出的买入价格区间，按**对齐规则**判定置信度：

| 对齐情况 | 置信度 | 定价策略 |
|:---------|:-----:|:---------|
| **4个维度指向同一区间**（最大偏差 < 15%） | ★★★★★ 极高 | 取4维加权均价作为**目标买入价** |
| **3个维度指向同一区间**（最大偏差 < 25%） | ★★★★☆ 高 | 取对齐的3个维度均值，第4个标注"偏离" |
| **2个维度指向同一区间**（2v2分歧） | ★★★☆☆ 中等 | 按股票类型权重高的维度优先，价格区间放宽±15% |
| **1个维度指向，其他3个分散** | ★★☆☆☆ 低 | 标注"极度分歧"，建议观望或仅轻仓试探 |
| **无任何维度重叠** | ★☆☆☆☆ 极低 | 不建议在当前价位买入，等待信号收敛 |

### 冲突解决规则（维度之间出现明显分歧时）

当2v2分歧（2个维度看多，2个维度看空）时，按以下**优先级判定**：

```
规则1：高分红股票（股息率>4%）→ 以维度3（股息率锚定法）为基准
规则2：周期股（行业明确）→ 以维度1（价格分布法）为基准
规则3：成长股（营收增速>20%）→ 以维度2（PE估值法）为基准，预期增速打8折
规则4：技术面无支撑 → 目标价下调一档（从稳健→激进，或从激进→放弃）
规则5：所有维度均指向不同方向 → 不发布目标价，标注"信号混乱，建议等待"
```

### 合成定价示例

```
格力电器（000651）— 高分红价值型
  维度1（价格分布）：理想L=36.89, 稳健=37.57~39.56, 激进=39.56~40.75
  维度2（PE估值）  ：EPS=1.09, 低估PE=10→10.9元, 合理PE=15→16.35元
  维度3（股息率）  ：DPS=2元, 激进4%→50元, 稳健5%→40元, 理想6%→33.33元
  维度4（技术面）  ：MA200=40.66（重要阻力位）, 52周最低=36.89（强支撑）

  对齐分析：
    - 📍 维度1稳健区间 37.57~39.56
    - 📍 维度3稳健区间 40元附近（5%股息率）
    - 📍 维度4支撑位 36.89元附近
    → 3个维度指向37~40元区间（最大偏差<8%）
    → 置信度：★★★★☆
    → 目标买入价：38~40元（加权均价）

  偏差说明：
    - 维度2（PE法）给出10.9~16.35元，严重偏离其他维度
    → 原因是格力EPS因分红后降低，PE法失效（每股净资产几乎全部被分红分掉了）
    → 忽略维度2，按3维对齐定价
```

### 输出要求（最终报告必须包含）

1. **股票类型判定**：明确标注判断依据（股息率/PE/行业）
2. **各维度计算表**：每个维度独立列出，含计算公式

```
| 维度 | 策略 | 买入价格 | 计算依据 |
|------|------|---------|---------|
| 价格分布 | 稳健 | 37.57~39.56元 | Q1(39.56)~中位数(40.75)下沿 |
| PE估值 | 不适用 | — | EPS因分红失真 |
| 股息率 | 稳健 | ~40元 | DPS=2元÷5%股息率 |
| 技术面 | 支撑 | 36.89元 | 52周最低价强支撑 |
```

3. **交叉验证汇总表**：
```
┌──────────────┬──────────┬──────────┬──────────┐
│              │ 维度1价格 │ 维度3股息 │ 维度4技术 │
│              │ 分布法    │ 率锚定法  │ 面验证   │
├──────────────┼──────────┼──────────┼──────────┤
│ 目标价格区间 │ 37.6~39.6│ ~40元    │ 36.89    │
│ 对齐情况     │ 对齐     │ 对齐     │ 对齐     │
│ 权重(高分红) │ 25%      │ 35%      │ 15%      │
└──────────────┴──────────┴──────────┴──────────┘
→ 加权目标价: 37.6×25% + 40×35% + 36.89×15% = 38.63元
→ 对齐度: 3维度一致 → ★★★★☆ 可信
```

4. **区分数据来源类型**：
   - **客观数据**（可精确计算）：PE倒推价格、DPS倒推价格、52周分位
   - **经验判断**（存在主观性）：PE合理区间选择、目标股息率设定、行业PE参考
5. **坦诚承认不确定性**："历史分位是统计概率，不代表未来价格一定在此区间内"

## 特殊情况处理

### 高成长股（营收增速>20%）
- PE估值法权重降低（成长股PE波动大）
- 增加 PEG 估值维度（PE / 增速 < 1 为低估）
- 股息率锚定法可能不适用（成长股通常低分红）

### 周期股（银行、钢铁、煤炭、猪企等）
- PE估值法不适用（周期股PE在景气周期高点反而低）
- 改用 PB 估值法（PB < 1 或接近历史最低 = 买入机会）
- 股息率锚定法依然适用

### 亏损股
- PE法不可用
- 改用 PS（市销率）或 PB 估值
- 重点关注收入增速、市场份额变化、扭亏预期

### 新股（上市<2年）
- 52周价格法参考价值有限
- 增加行业对标维度（同行业成熟公司估值对比）
- 提高风险权重
