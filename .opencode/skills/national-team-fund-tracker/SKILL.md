## 数据采集流程

```
Step 1: 调用 Tushare fund_share 获取每只ETF的日频份额
Step 2: 按月度分组，取每月最后一个交易日
Step 3: 汇总所有ETF月度总量
Step 4: 计算环比变化
Step 5: NAV交叉验证（确认份额变化是真实赎回还是份额拆分）
```

### 数据说明

| 字段 | 来源 | 单位 | 说明 |
|------|------|:----:|------|
| `fd_share` | Tushare `fund_share` | **万份** | 基金总份额（所有持有人之和） |
| `trade_date` | Tushare `fund_share` | — | 交易日 |
| `unit_nav` | Tushare `fund_nav` | 元/份 | 每份基金单位净值 |

**重要**: `fd_share` 是ETF在全市场的**总份额**，不是汇金的单一持有人持仓。份额变化可能部分来自其他持有人（保险、FOF、散户等）的申赎。

### 交叉验证方法

份额大幅变化时，需对照 `unit_nav` 判断：

| 现象 | 判断 |
|------|------|
| 份额骤降 + NAV基本不变 | ✅ **真实赎回** |
| 份额骤降 + NAV按比例翻倍 | ⚠️ **份额合并（反向拆细）**，非真实减持 |
| 份额稳定 + NAV变化 | 📉 正常净值波动 |

## 分析框架

### 月度趋势判断

获取逐月ETF份额总量后，按以下框架分析：

```
【Step 1: 数据加载】（必选）
   运行 main.py → 获取各ETF月末份额 + 汇总
   ⚠️ 询问用户是否生成HTML图表报告（可选，用户确认才执行）：
     report_html.py → output/national_team_etf_tracker.html

【Step 2: 总量趋势判断】（必选）
   判断维度：
   ① 当月环比变化率 = (当月总量 - 上月总量) / 上月总量
   ② 累计变化率 = (当前总量 - 基期总量) / 基期总量
   ③ 变化趋势（连续N月上升/下降/震荡）

【Step 3: 结构分析】（必选）
   判断维度：
   ① 沪深300 vs 科创50 vs 中证1000 的分化情况
   ② 科创50是否独立于大盘（反映汇金是否在调仓而非撤退）
   ③ 各ETF份额变化方向是否一致

【Step 4: NAV验证】（必选）
   对变化量 >5% 的月份，运行 NAV cross-check:
   ① 对比变化前后 unit_nav 是否稳定
   ② 确认是否为真实赎回

【Step 5: 综合判断】（必选）
   综合以上步骤给出结论
```

### 判定阈值

| 环比变化 | 信号 | 解读 |
|:--------:|:----:|:-----|
| > +5% | 🔵 **明显增持** | 国家队资金大规模入场，通常对应市场负面冲击后的维稳 |
| +1%~+5% | 🟢 **温和增持** | 正常定投或小幅加仓 |
| -1%~+1% | ⚪ **基本平稳** | 正常申赎范围内 |
| -5%~-1% | 🟡 **温和减持** | 小幅规模缩减，需关注后续方向 |
| < -5% | 🔴 **大规模赎回** | 国家队可能阶段性撤退或调仓，需验证是否为全市场趋势 |

### 结构信号

| 现象 | 信号 | 解读 |
|------|:----:|:------|
| 沪深300/上证50缩 + 科创50稳 | 🟡 **调仓** | 国家队可能从大盘转向科创，非全面撤退 |
| 所有ETF同步缩水 | 🔴 **全面撤退** | 国家队整体减仓 |
| 沪深300/上证50扩 + 科创50缩 | 🟢 **防御配置** | 国家队转入大盘防御 |
| 中证1000同步变化 | ℹ️ **市场共振** | 变化可能是全市场ETF趋势，非国家队单方面行为 |

## 分析工作流

### 数据采集（必选）

- [ ] 运行 `main.py` 采集数据（终端输出）
- [ ] 询问用户是否生成HTML图表报告
  - 是 → 运行 `report_html.py` → 展示HTML路径
  - 否 → 跳过

### 分析评估（必选，数据采集后立即执行，无需用户确认）

- [ ] **总量趋势** — 当月环比、累计变化、方向判断
- [ ] **结构分化** — 不同ETF方向是否一致
- [ ] **NAV验证** — 大幅变化月份验证赎回真实性
- [ ] **市场对比** — 对比非汇金ETF变化，判断是否为全市场趋势
- [ ] **综合判断** — 汇总以上维度给出国家队方向结论

## 注意事项

1. **份额≠持仓**：`fd_share` 是ETF总份额，汇金是最大持有人但不是唯一持有人
2. **季度持仓更精确**：汇金精确持仓只在基金半年报/年报的"前十名持有人"中披露
3. **市场共振**：全市场ETF同时缩水时，不一定是国家队行为
4. **ETF拆分/合并**：份额大幅变化时需验证NAV，排除数据口径变化
5. **数据延迟**：Tushare fund_share 数据可能有1-2个交易日的延迟
