# 第四步：综合评级

## 分析目标

综合前四步分析结果，输出最终评级和操作建议。

## 评级体系

| 评级 | 说明 | 条件 |
|------|------|------|
| A — 优秀 🟢 | 时间线完整，分析价值高 | 完整性≥80分 + 阶段结构合理 + 密度适中 |
| B — 良好 🟡 | 时间线基本完整，有一定分析价值 | 完整性≥60分 + 阶段结构可接受 |
| C — 一般 🟠 | 时间线不完整或噪声较多，分析价值有限 | 完整性≥40分 |
| D — 不足 🔴 | 事件严重缺失或噪声过多，不建议进一步分析 | 完整性<40分 |

## 判定规则

1. **完整性是硬门槛**：完整性评分低于对应等级的下限，即使其他维度正常，也不得升级
2. **阶段结构异常可降级**：出现"⚡预警信号"2 个以上，在完整性基础上降一级
3. **密度异常可调整**：密度极稀疏或过于密集 ±0.5 档
4. **综合评级不得高于完整性评分对应的等级**
