# 第三步：事件密度分析

## 分析目标

分析事件在时间轴上的分布密度，判断事件的活跃程度和节奏。

## 计算方法

**事件密度 = 事件总数 ÷ 时间跨度（月）**

## 密度参考值

| 密度（条/月） | 评级 | 含义 |
|---------------|------|------|
| > 2.0 | 🔴 非常密集 | 事件高度密集，可能包含大量冗余新闻 |
| 1.0 - 2.0 | 🟡 较密集 | 事件活跃期，节奏较快 |
| 0.3 - 1.0 | 🟢 适中 | 正常的事件节奏 |
| 0.1 - 0.3 | 🔵 稀疏 | 长期项目、边缘事件的典型节奏 |
| < 0.1 | ⚪ 极稀疏 | 几乎无有效追踪事件 |

## 进一步分析

### 活跃期识别

找出密度最高的 12 个月时间段，识别"活跃期"。

### 密度变化趋势

| 趋势 | 含义 |
|------|------|
| 前疏后密 | 事件正在加速推进 |
| 前密后疏 | 事件热度减退或进入尾声 |
| 均匀分布 | 按部就班推进 |
| 脉冲式 | 密集事件→长时间空白→再出现密集事件 |
