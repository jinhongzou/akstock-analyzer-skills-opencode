# 第〇步：逐事件影响评估

## 分析目标

对 HTML 时间线中的**每条事件**进行独立评估，判断其在整体事件演变中的角色和影响。

## 操作流程

1. **打开 HTML 报告**：在浏览器中打开 `output/{code}_{keyword}_chronos.html`
2. **点击链接**：点击每条事件的标题或 "🔗 原文" 按钮，阅读完整原始材料（巨潮 PDF 或东方财富新闻）
3. **逐条评估**：对每条事件按以下标准独立评估
4. **填入评估**：将评估结果填入 AI 评估区（见下方"程序化替换"）

---

## 事件角色分类

每条事件在整体发展过程中扮演的角色：

| 标签 | 说明 | 判断标准 |
|------|------|---------|
| `[触发信号]` | 标志事件开始或局面转折 | 事件本身改变了事态方向（如：首次提出、首次出现关键确认） |
| `[关键里程碑]` | 对进展有决定性影响的事件 | 没有此事件，后续不会发生（如：获批、签约、交接完成、条件成就） |
| `[常规进展]` | 按计划推进的例行更新 | 信息型公告、定期报告、程序性公告 |
| `[风险事件]` | 对事件产生负面冲击 | 不利消息、违约、诉讼、政策变化 |
| `[背景信息]` | 提供环境背景，非事件主线 | 行业政策变化、宏观环境、媒体报道 |

> 每一条事件必须且只能选择一个角色分类。

---

## 影响方向

评估该事件对整体事件发展**有利还是不利**：

| 方向 | 含义 | 示例 |
|------|------|------|
| `正面` | 推动事件朝预期方向前进 | 审批通过、资金到位、进度加速 |
| `负面` | 阻碍或延缓发展进程 | 审批被拒、资金紧张、诉讼 |
| `中性` | 无方向性影响 | 例行披露、信息说明 |
| `不确定` | 方向不明或尚未见分晓 | 新政策出台、董事会构成变化 |

---

## 影响程度

评估该事件对整体发展的**重要性级别**：

| 程度 | 含义 | 判断标准 |
|------|------|---------|
| `重大` | 对事件进程有决定性影响 | 事件直接改变发展路径或结果（如终止/获批/交割） |
| `中等` | 明显影响但非决定性的 | 事件明显推动或阻碍进展，但不改变根本方向（如大额合同/重要人事变动/大股东增持） |
| `轻微` | 影响有限或程序性的 | 例行性公告、小幅变动 |

---

## 综合判断规则

1. 先读原文再判断，不得仅凭标题猜测
2. 每条事件独立评估，**不应受前后事件的干扰**
3. 若原文信息不足（如公告仅一句话且无具体内容），标注 `[常规进展] 中性 - 轻微`

---

## 填写要求

### 格式

```
[事件角色] 一句话评估摘要。影响：影响方向 - 影响程度 - 关联说明
```

### 示例

```
[关键里程碑] 环评获批，经历近两年审批终于获得环保批复，是整个项目最重大的里程碑。影响：正面 - 重大 - 为后续建设扫清法律障碍，但时间成本已很高
```

```
[风险事件] 临时股东大会通知，审议多项借款及担保议案，暴露管理层融资压力。影响：负面 - 中等 - 大量借款增加财务费用，压缩公司利润空间
```

```
[常规进展] 发布关于重大资产重组进展公告，继续停牌。影响：中性 - 轻微 - 属于例行进展披露，无新信息增量
```

## 程序化替换说明

HTML 中每条事件的 AI 评估区使用**固定的占位符字符串**，完成逐事件评估后，通过 Python 替换填入。

### 占位符精确字符串

```
[AI评估待补充] 点击原文链接阅读后，在此填入你的分析
```

> 此字符串在**所有事件中完全相同**（不因关键词变化），可直接 `str.replace()` 全局匹配。

### 批量替换（所有事件相同评估）

```python
with open("output/600338_阿根廷_chronos.html", "r", encoding="utf-8") as f:
    html = f.read()

assessment = "[关键里程碑] 环评获批，是项目最重大的里程碑。影响：正面 - 重大"
html = html.replace(
    "[AI评估待补充] 点击原文链接阅读后，在此填入你的分析",
    assessment
)

with open("output/600338_阿根廷_chronos.html", "w", encoding="utf-8") as f:
    f.write(html)
```

### 逐事件替换（每条事件不同评估）

```python
with open("output/600338_阿根廷_chronos.html", "r", encoding="utf-8") as f:
    html = f.read()

assessments = {
    1: "[触发信号] 首次提出项目规划。影响：正面 - 重大 - 标志事件开端",
    2: "[关键里程碑] 环评获批。影响：正面 - 重大 - 扫清建设法律障碍",
    3: "[风险事件] 项目资金审批暂缓。影响：负面 - 中等 - 施工进度预期将延迟",
    # ... 按实际事件序号继续填写
}

placeholder = "[AI评估待补充] 点击原文链接阅读后，在此填入你的分析"

# ⚠️ 从右向左替换，确保 data-seq 顺序正确
for seq in sorted(assessments.keys(), reverse=True):
    idx = html.rfind(placeholder)
    if idx >= 0:
        html = html[:idx] + assessments[seq] + html[idx + len(placeholder):]

with open("output/600338_阿根廷_chronos.html", "w", encoding="utf-8") as f:
    f.write(html)
```
