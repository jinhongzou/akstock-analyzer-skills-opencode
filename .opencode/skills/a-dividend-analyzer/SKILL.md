# A股分红配送分析

**功能**: 获取A股历年分红配送详情，包括送转比例、现金分红、股息率、每股指标、关键日期等  
**数据源**: 东方财富 (akshare) - `stock_fhps_detail_em`

---

## 使用方式

**OpenCode 调用**:
```
skill(name="a-dividend-analyzer")
```

**命令行运行**:
```bash
python .opencode/skills/core/src/skills/a-dividend-analyzer/main.py <A股代码>
```

### 示例

```bash
python .opencode/skills/core/src/skills/a-dividend-analyzer/main.py 600519   # 贵州茅台
python .opencode/skills/core/src/skills/a-dividend-analyzer/main.py 000858   # 五粮液
python .opencode/skills/core/src/skills/a-dividend-analyzer/main.py 300073   # 当升科技
```

## ⭐ 你的评估任务：分红质量5维度评分卡（必须执行）

> main.py 只输出历年分红数据。你必须基于以下评分卡，从原始数据中读取数值，逐项评分。

### 评分卡（5维度，满分40分）

| # | 维度 | 数据来源（main.py具体行） | 判定条件 | 得分 |
|---|------|-------------------------|---------|:----:|
| 1 | **分红连续性** | `现金分红年份数: XX` | ≥ 20年 | 10 |
|   |      |         | 10-19年 | 8 |
|   |      |         | 5-9年 | 5 |
|   |      |         | 3-4年 | 3 |
|   |      |         | < 3年 | 0 |
| 2 | **股息率水平** | `平均股息率: X%`（或最近年股息率） | > 5%（高股息） | 10 |
|   |      |         | 3-5%（中等） | 7 |
|   |      |         | 1-3%（偏低） | 4 |
|   |      |         | < 1%（微息） | 1 |
| 3 | **分红稳定性** | 查看最近5年分红记录 `每期分红额` | 每年都分红且金额稳定或增长 | 10 |
|   |      |         | 每年都分红但金额有波动 | 7 |
|   |      |         | 偶尔跳过或大幅减少 | 3 |
|   |      |         | 不规律或中断 | 0 |
| 4 | **送转含金量** | `送股次数: X` / `转股次数: X` | 现金分红为主，送转适度（现金分红年份占总数 > 80%） | 5 |
|   |      |         | 现金分红为主但有多次送转 | 3 |
|   |      |         | 送转为主，现金分红少 | 1 |
|   |      |         | 无分红仅有送转 | 0 |
| 5 | **可维持性**（定性判断） | 结合每股收益和分红比例 | EPS 覆盖 DPS 有余（EPS ≥ DPS×1.5）且分红比例合理 | 5 |
|   |      |         | EPS 刚好覆盖 DPS | 3 |
|   |      |         | EPS < DPS（分红超越利润，不可持续） | 0 |

> **注意**：维度5需要你自己计算：分红比例 = 每股分红(DPS) / 每股收益(EPS)。如 DPS > EPS，说明公司在用过往积累分红，不可持续。

### 综合判定

| 总分（满分40） | 评级 | 含义 |
|:-------------:|:----:|:-----|
| **35-40分** | 🌟🌟🌟 **分红标杆** | 连续高分红+高股息+稳定，A股顶级分红标的 |
| **25-34分** | 🌟🌟 **分红优良** | 分红稳定有吸引力，值得关注 |
| **15-24分** | 🌟 **分红一般** | 有分红但不突出，需关注可持续性 |
| **0-14分** | ❌ **分红不足** | 分红不稳定或水平低，不适合股息策略 |

### 评分示例

```
输入数据（格力电器 000651）：
  现金分红年份数: 33 → 维度1得10分（≥20年）
  平均股息率: 约5%+ → 维度2得10分（>5%）
  近5年分红：每年均有且2025年每股3元 → 维度3得10分（稳定）
  送股16次/转股17次 vs 现金33次 → 维度4得5分（现金为主占比高）
  EPS 1.09元 vs DPS约2-3元（2025年）→ 维度5得0分（DPS > EPS，不可持续）
  ————————————————————————
  总分: 10+10+10+5+0 = 35/40 → 🌟🌟🌟 分红标杆
  ⚠️ 注意：2025年分红金额超过EPS，需关注每股收益能否跟上分红增速
```

### 送转类型说明

| 类型 | 说明 |
|------|------|
| 送股 | 用未分配利润转增股本，需缴税 |
| 转股 | 用资本公积金转增股本，不缴税 |
| 现金分红 | 直接向股东派发现金 |
